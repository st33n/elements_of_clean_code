== HVAD ER CLEAN CODE SÅ?

* Kode man kan læse og sige "Aha!"
* Kode som nogen føler ansvar for
* Minimalt antal klasser, metoder, funktioner m.v.
* Ingen duplikation
* Kode som er testet af automatiske tests
** Tests -> nemmere at lave ændringer => Levende Kode
